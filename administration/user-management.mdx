## Creating Users

```bash
# Create single user
ava users create \
  --email john@company.com \
  --name "John Doe" \
  --role data-analyst \
  --send-invite

# Bulk import from CSV
ava users import --file users.csv
```

## Role Management

<Tabs>
  <Tab title="Assign Roles">
    ```bash
    ava users assign-role \
      --user john@company.com \
      --role data-governance-officer
    ```
  </Tab>

  <Tab title="Create Custom Role">
    ```bash
    ava roles create \
      --name "Marketing Analyst" \
      --permissions "data:read,reports:create"
    ```
  </Tab>
</Tabs>
