## Key Metrics

```bash
# View system status
ava status

# Check service health
ava health check

# View performance metrics
ava metrics --service api --period 24h
```

## Alerts

Configure alerts for system issues:

```yaml
alerts:
  - name: High Error Rate
    metric: error_rate
    threshold: "> 5%"
    notify: ops-team@company.com

  - name: Low Disk Space
    metric: disk_usage
    threshold: "> 85%"
    notify: ops-team@company.com
```
