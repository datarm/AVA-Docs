## Before Contacting Support

Please have the following information ready:

<Steps>
  <Step title="Organization Details">
    Your organization name and AVA instance URL
  </Step>
  <Step title="Issue Description">
    Clear description of the problem
  </Step>
  <Step title="Steps to Reproduce">
    What actions led to the issue
  </Step>
  <Step title="Error Messages">
    Any error messages or screenshots
  </Step>
  <Step title="System Information">
    Browser version, OS, etc.
  </Step>
</Steps>

## Issue Severity Levels

- **P0 - Critical**: System down, data loss, security breach
- **P1 - High**: Major functionality broken, significant impact
- **P2 - Medium**: Feature not working as expected, moderate impact
- **P3 - Low**: Minor issue, cosmetic problem, question

## What to Include in Your Support Request

To help us assist you quickly, please include:

- Organization name and AVA instance URL
- Clear description of the issue
- Steps to reproduce the problem
- Any error messages or screenshots
- Browser and OS information (if relevant)
