## Suggested KPIs

- **Adoption Rate**: % of users actively using AVA
- **Engagement**: Sessions per user per week
- **Feature Usage**: Chat vs Tasks vs Knowledge Search
- **Time Savings**: Estimated hours saved per user
- **User Satisfaction**: Survey scores and feedback
- **Support Tickets**: Reduction in IT/support requests
