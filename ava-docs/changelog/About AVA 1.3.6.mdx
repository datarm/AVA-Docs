# Updates in AVA Julius 1.3.6

This update to the Beta release introduces several exciting new features

> For complete User Documentation, go to [AVA User Documentation](https://datarm.notion.site/AVA-User-Documentation-d522aab7765b4f4aaf38aeaf0186e66e).

## General

- **Logout**: Select the logout option from the settings menu to log out of AVA.

## Chat

- **Sharing Chats**: Click the share icon on a chat in session history to share your chat with others on your team.

- **Fine-tune AVA**: Tune your AVA response with the new Analytic/Balanced/Creative switch for different chat modes.
- **Stop Button**: Ask the wrong question? You can interrupt AVA by clicking the stop button in Chat.

[Chat Documentation](https://datarm.notion.site/AVA-Chat-7799bf2a84a842cb8ddc164303f4f41b)

## Tasks

- **New Look**: Upgraded Tasks now include a new look and feel, with more intuitive layout and improved navigation.

- **Stop Button**: Stop a response from AVA by clicking the stop button in the Task window.

[Tasks Documentation](https://datarm.notion.site/AVA-Tasks-b59f920b85884e9f98cb3c2cba14d9e4)

## Knowledge Search

- **Secure Search**: Securely search your organizations private knowledge bases using natural language queries powered by AI Vector Similartity Search!

[Knowledge Search Documentation](https://datarm.notion.site/AVA-Knowledge-Search-In-Continued-Development-9b4904139f224574a8d4995f2d0ca333)

## Workspace

- **New Look**: Workspace now has a new look and feel, with a more intuitive layout and improved navigation.

- **Formatting**: New Workspaces now have more formatting options.
- **Edit in Workspace**: You can still edit with markdown if you prefer.
- **File Viewer**: View files cited in Knowledge Search chats, or review documents before adding them to the chat/task from the search modal.

## Bug Fixes

1. Workspace content now saves in tasks correctly.

2. Adding too much text to an AVA chat previously broke chat. This is now resolved.
