# AVA 2.8.0

**Release Date:** October 10, 2025

## Overview

AVA 2.8.0 delivers enhanced integration capabilities, advanced data lineage visualization, and improved governance workflows. This release emphasizes enterprise connectivity and streamlined compliance operations.

---

## New Features

<AccordionGroup>
  <Accordion title="Enhanced Data Lineage Visualization">
    - **Column-level Lineage**: Trace data transformations at the column level across entire data pipelines
    - **Impact Analysis**: Instantly visualize downstream impacts of schema or data changes
    - **Interactive Lineage Graph**: Zoom, filter, and explore complex data flows with improved UI
    - **Lineage Export**: Export lineage diagrams as SVG, PNG, or interactive HTML
    - **Historical Lineage**: Track how data flows have changed over time
  </Accordion>

  <Accordion title="Microsoft 365 Integration Suite">
    - **SharePoint Connector**: Scan and classify documents in SharePoint Online
    - **Teams Integration**: Governance notifications and approvals directly in Microsoft Teams
    - **OneDrive Scanning**: Automatic discovery of sensitive data in OneDrive for Business
    - **Outlook Add-in**: Data classification and DLP warnings in email composition
    - **Power BI Integration**: Governance metadata for Power BI datasets and reports
  </Accordion>

  <Accordion title="Advanced Workflow Engine">
    - **Visual Workflow Designer**: Build complex governance workflows with drag-and-drop
    - **Conditional Logic**: Create sophisticated approval chains with branching logic
    - **SLA Management**: Define and track service level agreements for governance requests
    - **Workflow Templates**: Pre-built templates for common governance scenarios
    - **Parallel Approvals**: Support for concurrent approval paths
  </Accordion>

  <Accordion title="Data Quality Monitoring">
    - **Automated Quality Checks**: Schedule and run data quality validations
    - **Quality Metrics Dashboard**: Monitor completeness, accuracy, consistency, and timeliness
    - **Anomaly Detection**: AI-driven detection of data quality issues
    - **Quality Alerts**: Configurable notifications for quality threshold violations
    - **Remediation Tracking**: Track data quality improvement initiatives
  </Accordion>
</AccordionGroup>
