Access live data from enterprise applications:

<Tabs>

  <Tab title="CRM & Sales">

  <Tab title="Development Tools">

  <Tab title="Other Integrations">

```
summary table with Company, Deal Value, Close Date, Stage, and
```

```
with section references."
```

## Model Selection & AI Features

Choose the right AI model for each task.

### Available Models

<Tabs>
  <Tab title="Azure OpenAI">
    **Text & Reasoning:**
    - **GPT-5**: Latest flagship model for logic-heavy and multi-step tasks
    - **GPT-5 Pro**: Extended reasoning capabilities for complex analysis
    - **GPT-5 Codex**: Advanced coding model for development tasks
    - **GPT-5 Chat**: Natural, multimodal, context-aware conversations
    - **GPT-5 Mini/Nano**: Lightweight, cost-effective variants for speed

    **Image Generation:**
    - **GPT-image-1**: Latest image generation with improved quality and detail
    - **DALL-E 3**: High-quality image generation with natural/vivid styles
  </Tab>
  <Tab title="AWS Bedrock">
    - **Claude Opus 4.1**: Most intelligent model, industry leader for coding and agents
    - **Claude Sonnet 4.5**: Most intelligent for complex agents, 1M token context
    - **Claude Haiku 4.5**: Fast, cost-effective near-frontier intelligence
    - **Claude Sonnet 4**: Hybrid reasoning with instant or deep thinking modes
    - **Claude Opus 4**: Next-gen model for autonomous AI agents
  </Tab>
  <Tab title="GCP Vertex">
    - **Gemini 2.5 Pro**: Most advanced model with Deep Think mode for complex reasoning
    - **Gemini 2.5 Flash**: Best price-performance, rapid creative workflows
    - **Gemini 2.5 Flash-Lite**: Speed and cost-optimized variant
    - **Gemini Pro**: Standard model for general-purpose tasks
  </Tab>
</Tabs>

### Model Selection Criteria

Choose based on:
- **Task Complexity**: Use GPT-5, GPT-5 Pro, Claude Opus 4.1, or Gemini 2.5 Pro for complex reasoning and multi-step logic
- **Speed Requirements**: Use GPT-5 Nano, Claude Haiku 4.5, or Gemini 2.5 Flash-Lite for quick responses
- **Coding Tasks**: GPT-5 Codex or Claude Opus 4.1 excel at code generation and debugging
- **Context Window**: Claude Sonnet 4.5 (1M tokens) for large document analysis
- **Cost**: Balance performance needs with budget - GPT-5 Mini/Nano, Flash, and Haiku models offer best price-performance
- **Conversations**: GPT-5 Chat and Gemini 2.5 models excel at natural, context-aware dialogue
- **Image Generation**: GPT-image-1 for latest quality, DALL-E 3 for creative visual content

```
Format financial data in tables with currency symbols.

My working hours are 8am-6pm weekdays."
```

AVA applies these preferences to all conversations and tasks.

## Security & Privacy Features

Enterprise-grade security built in.

<AccordionGroup>
  <Accordion title="Delegated Permissions">
    AVA only accesses what the authenticated user can access - no elevated privileges
  </Accordion>
  <Accordion title="Data Residency">
    All data stays within your Azure tenant - never sent to external SaaS providers
  </Accordion>
  <Accordion title="Audit Logging">
    Complete audit trail of all user actions via Application Insights
  </Accordion>
  <Accordion title="Encryption">
    TLS 1.3 in transit, Azure-managed encryption at rest
  </Accordion>
  <Accordion title="Identity-Based Access">
    Microsoft Entra ID authentication with SAML/OAuth
  </Accordion>
</AccordionGroup>
