## Version 2.1.0 - October 2024

### New Features

- **Enhanced AI Classification**: Improved machine learning models with 98% accuracy for PII/PHI detection
- **Databricks Unity Catalog**: Full support for Unity Catalog integration
- **Advanced Risk Scoring**: New risk scoring algorithm with customizable weights
- **Automated Remediation**: Auto-fix capabilities for common policy violations

### Improvements

- **Performance**: 40% faster data scanning for large datasets
- **UI/UX**: Redesigned dashboard with improved navigation
- **API**: New bulk operations endpoints for data sources and policies
- **Search**: Enhanced semantic search with better relevance ranking

### Bug Fixes

- Fixed issue with Snowflake key-pair authentication
- Resolved memory leak in background workers
- Corrected timezone handling in scheduled reports
- Fixed SAML authentication with Azure AD

## Version 2.0.0 - September 2024

### Major Release

- **Complete Platform Redesign**: New modern UI/UX
- **AI-Powered Features**: Intelligent search, automated classification, anomaly detection
- **Compliance Frameworks**: Pre-built frameworks for GDPR, CCPA, SOC2, HIPAA
- **API v2**: New RESTful API with improved performance and documentation

### New Integrations

- Salesforce
- Snowflake
- Databricks
- PostgreSQL
- MongoDB
- AWS S3
- Azure Blob Storage
- Google Cloud Storage
