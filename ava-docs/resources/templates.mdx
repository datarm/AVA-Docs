## What You'll Find

Prompt templates optimized for common workflows:

### By Department
- **Sales**: Pipeline reviews, account research, proposal generation
- **Support**: Customer inquiries, knowledge base searches, case updates
- **Development**: Sprint planning, code reviews, documentation
- **Executive**: Meeting prep, status reports, decision support
- **Marketing**: Content creation, campaign analysis, event planning
- **HR**: Policy questions, onboarding, employee records

### By Feature
- **AVA Chat**: Effective conversation prompts
- **AVA Tasks**: Multi-step workflow templates
- **Knowledge Search**: Optimized search queries
- **Export**: Document generation templates
