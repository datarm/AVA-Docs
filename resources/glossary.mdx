## Key Terms (Preview)

### Core Concepts
- **@ Mentions**: Syntax for accessing connected data sources (e.g., @Calendar, @Mail, @Salesforce)
- **Delegated Permissions**: AVA operates with your user permissions, not elevated access
- **MCP (Model Context Protocol)**: Standardized integration layer for enterprise applications
- **RAG (Retrieval Augmented Generation)**: AI technique used in Knowledge Search
- **SuperPrompt**: Enhanced AI guidance toggle in AVA Tasks

### Features
- **AVA Chat**: Conversational AI interface with @ mention access to connected data
- **AVA Tasks**: Multi-step workflow automation with sequential execution
- **Knowledge Search**: Semantic search across SharePoint documents using RAG
- **Workspace**: Three-tab panel (Search, Notes, File Viewer) for enhanced productivity
- **Prompt Library**: Saved, reusable prompts shared across your organization

### Technical Terms
- **Corpus**: Collection of documents for Knowledge Search (plural: corpora)
- **Embeddings**: Vector representations of text for semantic search
- **pgVector**: PostgreSQL extension for vector storage
- **Container Apps**: Azure service hosting AVA backend and MCP servers
- **Entra ID**: Microsoft's identity platform (formerly Azure AD)
