## Endpoints

### List Data Sources

```http
GET /api/v1/data-sources
```

```bash
curl -X GET https://app.datarm.com/api/v1/data-sources \
  -H "Authorization: Bearer YOUR_API_KEY"
```

**Response:**
```json
{
  "data": [
    {
      "id": "ds_abc123",
      "name": "Production Salesforce",
      "type": "salesforce",
      "status": "connected",
      "last_scan": "2024-10-23T10:00:00Z",
      "assets_count": 1523
    }
  ]
}
```

### Create Data Source

```http
POST /api/v1/data-sources
```

```bash
curl -X POST https://app.datarm.com/api/v1/data-sources \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Production Snowflake",
    "type": "snowflake",
    "config": {
      "account": "your-account.snowflakecomputing.com",
      "warehouse": "COMPUTE_WH",
      "database": "PROD_DB"
    },
    "credentials": {
      "username": "ava_service",
      "password": "encrypted_password"
    }
  }'
```

### Update Data Source

```http
PATCH /api/v1/data-sources/{id}
```

### Delete Data Source

```http
DELETE /api/v1/data-sources/{id}
```

### Trigger Scan

```http
POST /api/v1/data-sources/{id}/scan
```

```bash
curl -X POST https://app.datarm.com/api/v1/data-sources/ds_abc123/scan \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"profile": "standard"}'
```
