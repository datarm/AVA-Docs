## Endpoints

### List Policies

```http
GET /api/v1/policies
```

### Create Policy

```http
POST /api/v1/policies
```

```bash
curl -X POST https://app.datarm.com/api/v1/policies \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "PII Protection Policy",
    "type": "access_control",
    "scope": {
      "classifications": ["PII", "PII_EMAIL"]
    },
    "rules": [{
      "type": "require_encryption",
      "enabled": true
    }]
  }'
```

### Update Policy

```http
PATCH /api/v1/policies/{id}
```

### Delete Policy

```http
DELETE /api/v1/policies/{id}
```

### Apply Policy

```http
POST /api/v1/policies/{id}/apply
```
