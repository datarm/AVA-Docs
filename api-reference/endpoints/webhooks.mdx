## Overview

Webhooks allow you to receive real-time notifications when events occur in AVA.

## Create Webhook

```http
POST /api/v1/webhooks
```

```bash
curl -X POST https://app.datarm.com/api/v1/webhooks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://your-app.com/webhooks/ava",
    "events": [
      "policy_violation",
      "high_risk_detected",
      "scan_completed"
    ],
    "secret": "your_webhook_secret"
  }'
```

## Webhook Payload

```json
{
  "event": "policy_violation",
  "timestamp": "2024-10-23T14:30:00Z",
  "data": {
    "policy_id": "pol_abc123",
    "policy_name": "PII Protection",
    "resource": "database.customers",
    "severity": "high",
    "violation_details": {
      "type": "unencrypted_data",
      "classification": "PII"
    }
  }
}
```

## Verify Webhook Signatures

```python
import hmac
import hashlib

def verify_webhook(payload, signature, secret):
    expected = hmac.new(
        secret.encode(),
        payload.encode(),
        hashlib.sha256
    ).hexdigest()
    return hmac.compare_digest(expected, signature)
```
