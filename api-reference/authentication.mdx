## API Keys

Generate API keys in the AVA dashboard:

<Steps>
  <Step title="Navigate to Settings">
    Go to **Settings** > **API Keys**
  </Step>
  <Step title="Generate Key">
    Click **Generate New Key** and provide a description
  </Step>
  <Step title="Copy Key">
    Copy the key immediately - it won't be shown again
  </Step>
  <Step title="Use in Requests">
    Include in Authorization header: `Bearer YOUR_API_KEY`
  </Step>
</Steps>

## Authentication Methods

```bash
# Bearer token authentication (recommended)
curl -X GET https://app.datarm.com/api/v1/data-sources \
  -H "Authorization: Bearer ava_sk_live_abc123"

# Basic authentication (legacy)
curl -X GET https://app.datarm.com/api/v1/data-sources \
  -u "username:api_key"
```

## Security Best Practices

<Warning>
  - Never commit API keys to version control
  - Use environment variables for keys
  - Rotate keys every 90 days
  - Use minimum required scopes
  - Set expiration dates
</Warning>
