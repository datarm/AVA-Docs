## Quick Start

<Tabs>
  <Tab title="Python">
    ```bash
    pip install ava-sdk
    ```

    ```python
    from ava import Client

    client = Client(api_key="YOUR_API_KEY")
    sources = client.data_sources.list()
    ```
  </Tab>

  <Tab title="JavaScript">
    ```bash
    npm install @datarm/ava-sdk
    ```

    ```javascript
    import { AVAClient } from '@datarm/ava-sdk';

    const client = new AVAClient({ apiKey: 'YOUR_API_KEY' });
    const sources = await client.dataSources.list();
    ```
  </Tab>

  <Tab title="Java">
    ```xml
    <dependency>
      <groupId>com.datarm</groupId>
      <artifactId>ava-sdk</artifactId>
      <version>1.0.0</version>
    </dependency>
    ```

    ```java
    import com.datarm.ava.Client;

    Client client = new Client("YOUR_API_KEY");
    List<DataSource> sources = client.getDataSources().list();
    ```
  </Tab>
</Tabs>
