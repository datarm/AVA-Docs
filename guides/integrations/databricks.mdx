## Overview

Integrate AVA with Databricks to provide comprehensive governance for your lakehouse data platform.

## Setup

<Steps>
  <Step title="Generate Access Token">
    In Databricks, go to **User Settings** > **Access Tokens** > **Generate New Token**
  </Step>
  <Step title="Connect in AVA">
    ```bash
    ava connect databricks \
      --name "Production Databricks" \
      --workspace-url "https://your-workspace.cloud.databricks.com" \
      --token "YOUR_ACCESS_TOKEN" \
      --catalog "main"
    ```
  </Step>
  <Step title="Verify Connection">
    ```bash
    ava connections test databricks
    ```
  </Step>
</Steps>

## Unity Catalog Integration

AVA integrates with Unity Catalog for enhanced governance:

```bash
ava connect databricks \
  --unity-catalog enabled \
  --metastore-id "your-metastore-id" \
  --sync-lineage true
```
