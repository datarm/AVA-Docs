## Configure Alert Rules

```bash
# Create alert rule
ava alerts create \
  --name "Critical Risk Alert" \
  --condition "risk_score > 90" \
  --channels "email,slack" \
  --recipients "security-team@company.com" \
  --slack-channel "#security-alerts"
```

## Alert Types

### Risk-Based Alerts

```yaml
alerts:
  - name: High Risk PII Detection
    trigger:
      risk_level: [critical, high]
      classification: [PII, PHI]
    channels: [email, pagerduty]
    priority: high

  - name: Compliance Violation
    trigger:
      policy_violation: true
      severity: critical
    channels: [email, slack]
    priority: critical
```

### Access-Based Alerts

```bash
# Alert on unusual access
ava alerts create-access-alert \
  --type "unusual_time" \
  --classification "SENSITIVE" \
  --channel slack \
  --webhook "https://hooks.slack.com/..."
```
