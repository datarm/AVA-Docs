## Overview

AVA's intelligent search leverages natural language processing to help you find data using plain English queries instead of complex SQL or filter syntax.

## Natural Language Queries

Ask questions in plain English:

```bash
# Natural language search
ava search "Find all customer tables with email addresses"
ava search "Show me PII data in Salesforce"
ava search "What databases contain financial information?"
```

## Semantic Search

AVA understands context and intent:

<Tabs>
  <Tab title="Python">
    ```python
    from ava import Client

    client = Client(api_key="YOUR_API_KEY")

    # Semantic search
    results = client.search.semantic(
        query="customer contact information",
        sources=["salesforce", "snowflake"]
    )

    for result in results:
        print(f"{result.name}: {result.relevance_score}")
    ```
  </Tab>

  <Tab title="CLI">
    ```bash
    # Semantic search
    ava search semantic \
      --query "customer contact information" \
      --sources "salesforce,snowflake" \
      --limit 20
    ```
  </Tab>
</Tabs>

## Advanced Search

Combine natural language with filters:

```python
# Advanced search
results = client.search.advanced(
    query="customer data",
    filters={
        "classification": ["PII", "SENSITIVE"],
        "risk_score": {"min": 70},
        "last_accessed": {"within": "30d"}
    },
    sort_by="relevance"
)
```
