### Connected Data Sources

Access live data from enterprise applications:

<Tabs>
  <Tab title="Microsoft 365">
    - **@Mail**: Query your Outlook inbox and sent items
    - **@Calendar**: Check schedule, find meetings, see availability
    - **@Files**: Search and access OneDrive and SharePoint documents
    - **@Teams**: Access Teams messages and channel information
    - **@ToDo**: View and manage your task lists
  </Tab>
  <Tab title="CRM & Sales">
    - **@Salesforce**: Query opportunities, leads, cases, accounts, contacts
    - Update CRM records directly from chat
    - Create new records with AI assistance
    - Generate reports and forecasts
  </Tab>
  <Tab title="Development Tools">
    - **@GitHub**: Search repositories, check PR status, review code
    - **@Jira**: Query issues, update tickets, check sprint status
    - Link commits to conversations
    - Generate development summaries
  </Tab>
  <Tab title="Other Integrations">
    - **@NetSuite**: Business records and transactions
    - **@Security**: Microsoft Defender alerts and incidents
    - **@Web**: Web scraping and content crawling
    - **Custom MCP servers**: Extend with your own integrations
  </Tab>
</Tabs>

### Conversation Capabilities

- **Context Awareness**: AVA remembers earlier in the conversation and maintains context across @ mentions
- **Multi-Turn Dialogues**: Natural back-and-forth conversations with follow-up questions
- **Code Generation**: Generate and explain code in multiple languages
- **Data Analysis**: Analyze data from connected sources and generate insights
- **Content Creation**: Draft emails, documents, presentations with business context

### Example Prompts

```
Name: "Weekly Sales Pipeline Review"
Connected Data: @Salesforce
File Types: Excel
Sharing: Sales Team

Prompt: "Review all opportunities closing this week. Create a
summary table with Company, Deal Value, Close Date, Stage, and
Risk Assessment. Sort by deal value descending. Export to Excel."
```

```
Name: "Contract Clause Extraction"
Connected Data: @Contracts Demo (Knowledge Search)
File Types: PDF, Word
Sharing: Legal Team

Prompt: "Extract all payment terms, termination clauses, and
renewal provisions from this contract. Create a structured summary
with section references."
```

## Model Selection & AI Features

Choose the right AI model for each task.

### Available Models

<Tabs>
  <Tab title="Azure OpenAI">
    - **GPT-4**: Best for complex reasoning and analysis
    - **GPT-4 Turbo**: Faster GPT-4 with larger context window
    - **GPT-3.5 Turbo**: Fast, cost-effective for simpler tasks
    - **o1/o3-mini**: Optimized models for specific use cases
  </Tab>
  <Tab title="AWS Bedrock">
    - **Claude Sonnet**: Balanced performance and cost
    - **Claude Opus**: Highest capability Claude model
    - **Claude Haiku**: Fast, cost-effective Claude variant
  </Tab>
  <Tab title="GCP Vertex">
    - **Gemini Pro**: Google's advanced multimodal model
    - **PaLM 2**: Text generation and analysis
  </Tab>
</Tabs>

### Model Selection Criteria

Choose based on:
- **Task Complexity**: Use GPT-4 or Claude Opus for complex analysis
- **Speed Requirements**: Use GPT-3.5 or Claude Haiku for quick responses
- **Context Window**: GPT-4 Turbo for large document analysis
- **Cost**: Balance performance needs with budget considerations
- **Specific Capabilities**: Some models excel at code, others at creative writing

### Custom Instructions

Personalize AVA's behavior globally:

```
Example Custom Instructions:

"I'm a VP of Sales at a B2B SaaS company. When analyzing
opportunities:
- Always show ARR (Annual Recurring Revenue)
- Include sales cycle length
- Highlight competitive threats
- Flag deals over $100K

Format financial data in tables with currency symbols.
My timezone is Pacific Time (PT).
My working hours are 8am-6pm weekdays."
```

AVA applies these preferences to all conversations and tasks.

## Security & Privacy Features

Enterprise-grade security built in.

<AccordionGroup>
  <Accordion title="Delegated Permissions">
    AVA only accesses what the authenticated user can access - no elevated privileges
  </Accordion>
  <Accordion title="Data Residency">
    All data stays within your Azure tenant - never sent to external SaaS providers
  </Accordion>
  <Accordion title="Audit Logging">
    Complete audit trail of all user actions via Application Insights
  </Accordion>
  <Accordion title="Encryption">
    TLS 1.3 in transit, Azure-managed encryption at rest
  </Accordion>
  <Accordion title="Identity-Based Access">

</AccordionGroup>
