## AI-Powered Classification

Intelligent data classification with machine learning:

- **Sensitive Data Detection**: Automatically identify PII, PHI, PCI, and financial data
- **Custom Categories**: Define your own classification tags and rules
- **Confidence Scores**: ML-based confidence levels for each classification
- **Bulk Classification**: Classify thousands of assets simultaneously
- **Manual Override**: Review and adjust AI classifications as needed
