## Overview

AVA helps organizations comply with the General Data Protection Regulation (GDPR) through automated data discovery, classification, and governance.

## AVA GDPR Features

### Data Discovery
Automatically identify personal data across all systems.

### Data Subject Requests (DSAR)
```bash
# Process access request
ava dsar create \
  --type access \
  --subject-email "john@example.com" \
  --export-format json

# Process erasure request
ava dsar create \
  --type erasure \
  --subject-email "john@example.com" \
  --verify-deletion
```

### Consent Management
Track and manage consent records for data processing.

### Audit Trails
Complete audit logs for all data access and processing activities.

## Compliance Dashboard

View GDPR compliance status:
```bash
ava compliance status --framework gdpr
```
