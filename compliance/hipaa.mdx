## PHI Protection

### Automatic PHI Detection
AVA automatically identifies and classifies PHI:
- Patient names and IDs
- Medical record numbers
- Health plan numbers
- Diagnoses and treatments
- Biometric data

### Encryption
```bash
# Enable encryption for PHI
ava compliance configure-hipaa \
  --encrypt-phi true \
  --encryption-algorithm aes-256 \
  --key-rotation 90d
```

### Access Controls
- Role-based access to PHI
- Minimum necessary access
- Emergency access procedures
- Automatic session timeout

### Audit Logging
Complete audit trails for all PHI access:
```bash
# View PHI access logs
ava audit phi-access \
  --days 30 \
  --export hipaa-audit-log.pdf
```

## Breach Notification

Automated breach detection and notification workflow:
1. Detect unauthorized PHI access
2. Assess breach severity
3. Notify affected parties
4. Report to HHS if required
